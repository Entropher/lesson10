# Lesson 10 — HTML Forms & Meta Tags

A static webpage built as a front-end course homework assignment.
Part of a web development course. The page simulates a construction/real estate company landing page (SKY HOUSE) written in Georgian, with Lorem Ipsum body content and a working contact form layout.

## What it covers

- Semantic HTML structure (header, main, footer, article, section)
- Meta tags (`charset`, `viewport`, `description`)
- An HTML form with text, email, tel, and textarea inputs
- An embedded OpenStreetMap iframe
- External CSS via a linked stylesheet
